## 💡 Tips & Tricks

### For Students
- **Click symptom cards** to select (they get colorful shadows!)
- **Choose severity** by clicking Mild/Moderate/Severe buttons
- **Select location** from dropdown
- **Submit** and watch the success animation!

### For Admins
- **Hover over charts** to see detailed data
- **Click sections** in sidebar to jump to different areas
- **Export data** button downloads JSON file
- **All stats auto-calculate** from reports

### For Testing
- **No data yet?** Admin dashboard auto-generates 30-40 sample reports!
- **Want fresh start?** Open console and type: `localStorage.clear()`
- **See colors?** Check out `color-guide.html`!

## 🚨 Troubleshooting

### Charts not showing?
- Make sure you're online (Chart.js loads from CDN)
- Check browser console for errors

### No data on admin dashboard?
- It auto-generates mock data on first load
- Try refreshing the page

### Colors not vibrant?
- Use Chrome, Firefox, Safari, or Edge (latest versions)
- IE 11 has limited gradient support

### Login not working?
- ANY email/password works (demo mode!)
- Check browser console for JavaScript errors
