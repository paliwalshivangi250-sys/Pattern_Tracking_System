  Error:  ImportError: No module named 'sklearn'

  ────────────────────────────────────────────────────────────────
  Error:  "method": "Z-Score (Fallback)"
  Fix:    1. Seed database: curl -X POST localhost:5000/api/seed-database
          2. Restart server: python app.py
          3. Verify data: curl localhost:5000/api/admin/stats
  
  
  PROBLEM 3: "Insufficient data" errors
  ────────────────────────────────────────────────────────────────
  Error:  "reason": "Insufficient data"
  Fix:    1. Need 30+ reports: curl -X POST localhost:5000/api/seed-database
          2. Check count: sqlite3 data/pattern_tracking.db "SELECT COUNT(*) FROM reports;"
  
  
  PROBLEM 4: Connection refused
  ────────────────────────────────────────────────────────────────
  Error:  curl: (7) Failed to connect
  Fix:    1. Check server running: ps aux | grep "python app.py"
          2. Start server: python backend/app.py
          3. Check port: lsof -i :5000
