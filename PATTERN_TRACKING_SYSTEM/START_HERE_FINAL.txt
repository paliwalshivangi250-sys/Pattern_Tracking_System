┌─────────────────────────────────────────────────────────────────────────┐
│  🐛 TROUBLESHOOTING                                                     │
└─────────────────────────────────────────────────────────────────────────┘

  Problem: Blank ML Charts
  ───────────────────────────────────────────────────────────────────────
  Solution:
  1. Check backend is running (python backend/app.py)
  2. Check browser console for errors (F12)
  3. Verify: http://localhost:5000/api/admin/ml/advanced-analytics

  Problem: Charts Show Mock Data
  ───────────────────────────────────────────────────────────────────────
  Solution:
  1. Backend not running → start it
  2. Check CORS settings in backend/config.py
  3. Seed database: curl -X POST localhost:5000/api/seed-database

  Problem: Import Errors
  ───────────────────────────────────────────────────────────────────────
  Solution:
  1. Install dependencies: pip install -r backend/requirements.txt
  2. Check Python version: python --version (need 3.8+)
  3. Try in virtual environment
